# "150 жилийн хатаалт" Монгол мод хатаах технологи
"150 жилийн хатаалт" нэртэй Монгол мод хатаах технологи. Энэхүү технологи нь модыг 150 жилийн турш хадгалсан модтой адилтгахуйц шинж чанартай мод бэлтгэх арга юм.

## Хөгжим урлаачид, хөгжмийн зэмсэгийн багаж, материалын худалдаа эрхлэгчид өгөх зөвлөгөө
Хөгжмийн зэмсгийн мод нь бусад модтой адил бөгөөд хадгалсан хугацаандаа бус харин бүрэн сайн, жигд хатаасан байх шаардлагатай байдаг. Технологийн дагуу хатаалга хийгдсэн модны хатуулаг шинж, дуу авиа ойлгох чадвар болон насжилт уртсах боломжтой.

Тэгвэл худалдаа эрхлэгчид, хөгжим урлаачид модоо хатаан бэлтгэхэд өндөр үнэтэй тусдаг ба импортын мод ашиглах нь зардлыг нэмэгдүүлдэг. 

## Асуудал
Үйлдвэрлэлийн аргаар модыг хатаахад өндөр халуунд 8-12 цагийн турш зогсолтгүй хатаах хэрэгтэй болдог ба цахилгаан, түлшний асар өндөр зардалтай байдаг.

## Шийдэл
Мод хатаах технологийг хөгжим урлаач, хөгжмийн багаж материалын худалдаа эрхлэгчдийн өдөр тутмын амьдралын нэг зан үйл болгосноор модон материалын үнийг үлэмж хэмжээгээр бууруулах боломжтой.
Монголчуудын тэр дундаа Улаанбаатарчуудын хувьд орон сууцанд амьдрах нь элбэг бөгөөд төвийн халаалттай байдаг. Өвөлдөө гэрийнхээ паррыг ашиглан өвлийн турш модоо хатаан бэлтгэвэл:
1. Урт хугацааны турш бүрэн хатаалга хийгдэнэ.
2. Хатаан технологийн зардал 0₮ буюу ямар нэг тог цахилгаан, шатаах зуух ашиглахгүй.
3. Зориулалтын дагуу бэлдсэн төмөр хавтан шахагч төхөөрөмжийн нэг удаагийн зардал гарна.

Дээрх технологийг бүгд хэрэглэж заншвал хөгжмийн дуугаралт, чанар хийцийг нэмэгдүүлэхээс гадна модны үнийг бууруулж хөгжмийн багаж, материалын худалдаа эрхлэгч, хөгжим урлаач нар өндөр ашигтай ажиллах боломжтой. 

## Prototype 
Хөгжим хийх модыг удаан хугацааны хатаалганд зориулан шахах төхөөрөмж
зориулалт:
- Модонд тодорхой хэмжээний шахалт үүсгэх. Энэ нь модны ширхэг бүрийг буюу бүтцийг жигд болгох зорилготой.
- Хатаалтын турш модыг эргэх, мушгирах зэрэг гажилтаас сэргийлэн, тэгш, цэвэр хатаах
